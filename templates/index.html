Help & Instructions

Instructions:

1. Select plot type (Circular or Rectangular)

2. Adjust PPM range to focus on specific regions

3. Set saturation range for metabolite highlighting

4. Select metabolites to highlight in the plot

5. Click "Generate Plot" to update visualization

6. Use "Reset View" to restore all defaults

Zoom Controls:

• Mouse wheel: Zoom in/out

• Drag: Pan when zoomed

• Buttons: +/- zoom, ⌂ reset zoom

• Touch: Pinch to zoom, drag to pan

• Ctrl+R: Fast reset to defaults
